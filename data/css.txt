https://developer.mozilla.org/en-US/docs/Web/CSS, CSS - MDN Web Docs , CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG MathML or XHTML). CSS describes how elements should be rendered on screen on paper in speech or on other media. It can control the layout of multiple web pages all at once. CSS is among the core technologies of the web alongside HTML and JavaScript.
